# DemoExam
Приложение к экзамену 27 июня
Вариант 1

## Функционал

- Просмотр списка партнеров
- Добавление нового партнера
- Редактирование данных партнера
- Расчет скидки в зависимости от объема продаж
- Навигация между окнами (вперед/назад)

## Технологии

- .NET Framework
- Entity Framework
- SQL Server

## Запуск

1. Клонировать репозиторий
2. Восстановить пакеты NuGet
3. Настроить строку подключения к БД в App.config
4. Запустить проект

## Структура

- `MainWindow` - главный список партнеров
- `EditWindow` - форма добавления/редактирования
- `Models.cs` - модель данных
